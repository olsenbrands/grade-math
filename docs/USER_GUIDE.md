# Grade Math User Guide

## Welcome to Grade Math

Grade Math is an AI-powered homework grading app designed for teachers. Upload or scan student math homework, and get instant grading with detailed feedback.

---

## Getting Started

### 1. Create an Account

1. Visit the Grade Math website
2. Click "Sign Up"
3. Enter your email and create a password
4. Check your email and click the verification link
5. Complete your profile (name, school, grade level)

### 2. Understand Tokens

Grade Math uses tokens to manage AI grading:
- You start with **50 free tokens**
- Each graded submission costs **1 token**
- Adding feedback costs **1 additional token**
- Grading 10+ submissions gets a **10% discount**

Your token balance is always visible in the header.

---

## Creating a Project

Projects help you organize assignments.

### Steps:

1. Click "New Project" from the dashboard
2. Enter the assignment details:
   - **Title**: e.g., "Chapter 5 Quiz"
   - **Description**: What the assignment covers
   - **Due Date** (optional)
3. Click "Create Project"

### Adding an Answer Key

1. Open your project
2. Click "Add Answer Key"
3. Either:
   - Upload an image of the answer key
   - Manually enter answers
4. Save the answer key

---

## Uploading Student Work

### On Desktop (Drag & Drop)

1. Open your project
2. Drag homework files onto the upload zone
3. Or click "Browse Files" to select files
4. Supported formats: JPG, PNG, PDF, HEIC

### On Mobile (Camera)

1. Open the Grade Math app
2. Go to your project
3. Tap "Scan Homework"
4. Use the camera to capture each page
5. The alignment guide helps you frame the image
6. Tap capture (or let auto-capture work)

### Batch Scanning

1. Enable "Batch Mode"
2. Place each homework page in front of the camera
3. The app automatically captures when stable
4. Review captured images
5. Submit when done

---

## Grading Submissions

### Starting the Grading Process

1. Open your project
2. Select submissions to grade (or "Select All")
3. Click "Grade Selected"
4. Choose whether to include feedback
5. Confirm the token cost
6. Wait for processing

### Understanding Results

Each graded submission shows:
- **Score**: Points earned / Total points
- **Percentage**: Overall score percentage
- **Question Breakdown**: Each question with student/correct answers
- **Confidence**: How confident the AI is in the grading

### Confidence Indicators

- **Green (High)**: Very confident in the result
- **Yellow (Medium)**: May need review
- **Red (Low)**: Definitely needs teacher review

## Managing Students

### Adding Students

1. Go to "Students" in the navigation
2. Click "Add Student"
3. Enter the student's name
4. Click Save

### Auto-Assignment

When you upload homework, Grade Math tries to:
1. Detect the student's name on the paper
2. Match it to your roster
3. Auto-assign high-confidence matches

Low-confidence matches are flagged for your review.

### Manual Assignment

1. Open a submission
2. Click "Assign Student"
3. Select from your roster or add a new student

---

## Viewing Results

### Individual Results

1. Go to your project
2. Click on a submission
3. See the detailed grading result
4. Review each question
5. Make corrections if needed

### Class Summary

1. Open your project
2. Click "Summary" tab
3. See:
   - Class average
   - Score distribution
   - Completion status
   - Individual student scores

### Sharing Feedback

1. Open a grading result
2. Click "Share Feedback"
3. Options:
   - Copy link for digital sharing
   - Print for paper handouts

## Tips for Best Results

### Photo Quality

- Ensure good lighting
- Keep the camera steady
- Frame the entire page
- Avoid shadows and glare

### Answer Keys

- Make sure answers are clearly written
- Number questions consistently
- Include point values if partial credit applies

### Student Names

- Ask students to write names clearly
- Use consistent name formats
- The more submissions, the better auto-matching works

## Troubleshooting

### "Low Confidence" Results

The AI couldn't grade with certainty. Common causes:
- Unclear handwriting
- Smudged or crossed-out answers
- Missing answer key

**Solution**: Review and manually confirm the grade.

### "No Name Detected"

The AI couldn't find a name on the submission.

**Solution**: Manually assign the student.

### "Not Enough Tokens"

You've run out of grading tokens.

**Solution**: Purchase more tokens or wait for monthly refresh.

### Upload Failed

The file couldn't be uploaded.

**Possible causes**:
- File too large (max 10MB)
- Unsupported format
- Network issues

**Solution**: Try a smaller file or different format.
